Building an API Gateway with Spring Cloud - Part 3 (Zuul)

Date ven. 02 f√©vrier 2018 Modified mer. 31 janvier 2018 Tags Spring / Spring Cloud / Java

This post walks you through installing an embedded Zuul which is part of Spring Cloud developed by netflix OSS (Open Source Software). Zuul will use Eureka server in order to know how and where to route incoming requests and Ribbon to forward requests to the microservice. Ribbon is a HTTP client which allows client side load balancing.

  • Create the directory structure
$ mkdir -p zuul/src/main/java/zuul
$ mkdir -p zuul/src/main/resources
$ cd zuul
  • Create a Gradle build file
$ nano build.gradle

buildscript {
  repositories {
    mavenCentral()
  }
  dependencies {
    classpath("org.springframework.boot:spring-boot-gradle-plugin:1.5.10.RELEASE")
  }
}


apply plugin: 'java'
apply plugin: 'org.springframework.boot'
apply plugin: 'io.spring.dependency-management'

jar {
    baseName = 'gateway-zuul'
    version = '1.0'
}
sourceCompatibility = 1.8
targetCompatibility = 1.8

repositories {
    mavenCentral()
}

dependencies {
    compile('org.springframework.cloud:spring-cloud-starter-eureka')
    compile('org.springframework.cloud:spring-cloud-starter-zuul')
    compile('org.springframework.boot:spring-boot-starter-web')
}

dependencyManagement {
  imports {
    mavenBom 'org.springframework.cloud:spring-cloud-dependencies:Edgware.RELEASE'
  }
}
  • Create application
$ nano src/main/java/zuul/GatewayZuulApplication.java

package zuul;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;
import org.springframework.cloud.netflix.zuul.EnableZuulProxy;

@EnableZuulProxy
@SpringBootApplication
public class GatewayZuulApplication {

  public static void main(String[] args) {
    SpringApplication.run(GatewayZuulApplication.class, args);
  }

}
  • Configure Zuul
$ nano src/main/resources/application.yml

spring:
  application:
    name: zuul

ribbon: 
  ConnectTimeout: 30000
  ReadTimeout: 990000
  maxAutoRetries: 1
  MaxAutoRetriesNextServer: 2
  eureka:
    enabled: true

server: 
  port: 8080

zuul: 
  ignoredServices: '*'
  routes:
    api:
      path: /api/**
      serviceId: api
      sensitiveHeaders: Authorization

eureka:
  client:
    serviceUrl:
      defaultZone: http://localhost:8761/eureka/
  • Build
$ gradle build

  • Start Eureka and Microservice instances

  • Run

$ java -jar build/libs/gateway-zuul-1.0.jar
  • List routes
$ curl http://localhost:8080/routes
  • Test API access
curl http://localhost:8080/api/greeting

{"id":8,"content":"Hello, World!"}

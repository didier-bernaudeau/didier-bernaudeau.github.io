Laptop - Lenovo ThinkPad X240

Date ven. 12 juin 2015 Modified dim. 14 juin 2015 Tags Hardware

This article describes the configuration of my Lenovo ThinkPad X240 (20ALCT01WW) based on Intel Haswell processor. Future articles will refer to this one.

Devices

  • Processor: Intel i4300u (Haswell)
  • Memory : 8 Go DDR3L PC3-12800 1600 MHz
  • Graphics Controller: Intel HD Graphics 4400
  • Screen: 12.5" FHD (1920 x 1080) LED Backlit AntiGlare IPS Display
  • Audio controller : Intel HD Audio
  • Intel Ethernet Connection I218 LM
  • Intel Wireless 7260
  • Hard Disk Drive: HGST 500 Go replaced by an SDD Samsung 840 Pro 256 Go
  • WebCam: Lite-On Integrated Camera (USB)
  • SD card Reader: Realtek RTS5227 (PCI Express)
  • Smartcard Reader : Alcor Micro AU9540 (USB)
  • Fingerprint Reader : Validity Sensors (USB)
  • NFC Reader : Broadcom (SMbus)

Controller

  • PCI Express
  • USB xHCI (USB3)
  • USB EHCI (USB2)
  • SATA
  • LPC (Low Pin Count)
  • SMbus (System Management Bus)
  • HECI (Host Embedded Controller Interface)

UEFI BIOS

Version

  • Package version: 2.30 (GIUJ18US)
  • BIOS / UEFI version: 2.30 (GIET80WW)
  • ECP (Embedded Controller Programm) version: 1.13 (GIHT29WW)

configuration

Menu "Security / Virtualization"

Intel (R) Virtualization Technologie [Enabled]
Intel (R) VT-d Feature               [Enabled]

Menu " Security / Secure Boot"

Secure Boot                         [Disabled]

Menu "Startup"

UEFI / Legacy Boot                 [UEFI Only]
-CSM support                              [No]
Boot Mode                              [Quick]

HowTo make a web site with Pelican hosted on GitHub Pages

Date sam. 16 mai 2015 Modified mer. 08 juillet 2015 Tags Pelican / Gentoo

Pelican is a static site generator writed in Python. This means that you edit the content on your computer with a simple text editor. Then, a generator makes all HTML pages for the web site, including indexes for articles, categories and tags. In the end, your web site can be hosted easily on a simple HTTP server like GitHub Pages or Amazon S3.

Unlike other CMS (Content Management System) like Wordpress or Drupal, Pelican has no web administration interface. In most cases, the administration interface has many vulnerabilies used to deface the website.

This article describes step by step how to make your web site with Pelican 3.5 on Gentoo Linux and how to host your web site on GitHub Pages.

Install Pelican

# emerge pelican

Create GitHub space

On GitHub, create an account named "didier-bernaudeau" then a repository named "didier-bernaudeau.github.io". Obviously, remplace "didier-bernaudeau" by your username.

DNS Configuration

Add a record in the DNS :

Type : CNAME
Name : didier.bernaudeau.net
Value : didier-bernaudeau.github.io.

Check configuration with "dig" tool :

# emerge bind-tools

$ dig didier.bernaudeau.net +nostats +nocomments +nocmd

; <<>> DiG 9.10.1-P1 <<>> didier.bernaudeau.net +nostats +nocomments +nocmd
;; global options: +cmd
;didier.bernaudeau.net.         IN      A
didier.bernaudeau.net.  10800   IN      CNAME   didier-bernaudeau.github.io.
didier-bernaudeau.github.io. 3600 IN    CNAME   github.map.fastly.net.
github.map.fastly.net.  30      IN      A       185.31.17.133

Create local directory

$ git clone https://github.com/didier-bernaudeau/didier-bernaudeau.github.io.git/ MyWebSite
$ cd MyWebSite

Create project

$ pelican-quickstart

Download the theme

For my web site, I selected the theme "Bootstrap3" and the style sheet "Bootswatch".

$ cd MyWebSite
$ git clone https://github.com/DandyDev/pelican-bootstrap3.git/ theme
$ cd theme/static/css/
$ wget https://bootswatch.com/bower_components/bootstrap/dist/css/bootstrap.min.css
$ wget https://bootswatch.com/bower_components/bootstrap/dist/css/bootstrap.css

Moreover, I have edited the theme to change the font size of "blockquote" section (bootstrap.flatly.min.css).

Download plugins

$ cd MyWebSite
$ git clone --recursive https://github.com/getpelican/pelican-plugins plugins

Install Bootstrapify

Bootstrapify is a pelican plugin that modify article and page html to use bootstrap's default classes, especially tables. Beautiful Soup is requiered for Bootstrapify.

# emerge dev-python/beautifulsoup

Create GitHub Files

$ cd MyWebSite
$ mkdir content/GitHubFiles

Write your domain name in CNAME Files :

$ nano content/GitHubFiles/CNAME

didier.bernaudeau.net

Disable indexation by web search engine :

$ nano content/GitHubFiles/robots.txt

User-agent: *
Disallow:/

Pelican Configuration

$ nano pelicanconf.py

# Hide Blogroll
#LINKS =

# Hide Social widget
#SOCIAL =

# Define the theme
THEME = 'theme/'
BOOTSTRAP_THEME = 'flatly'

# Define menu options
DISPLAY_CATEGORIES_ON_MENU = True
DISPLAY_PAGES_ON_MENU = True

# Define index options
DISPLAY_ARTICLE_INFO_ON_INDEX = True
SUMMARY_MAX_LENGTH = 50

# Add description about author
ABOUT_ME = '...'

# Define sidebar options (With cloud tag)
DISPLAY_TAGS_ON_SIDEBAR = True
DISPLAY_TAGS_INLINE = True

# Add Creative Common licence in the footer
CC_LICENSE = 'CC-BY-NC-ND'

# Add plugins (Summary)
PLUGIN_PATHS = ['plugins']
PLUGINS = ['summary','bootstrapify']

# Add static paths will be copied without parsing their contents
STATIC_PATHS = ['images','presentation', 'GitHubFiles']

# Shift the installed location of a file
EXTRA_PATH_METADATA = {
   'GitHubFiles/CNAME': {'path': 'CNAME'},
   'GitHubFiles/robots.txt': {'path': 'robots.txt'},
}

Create an article in reStructuredText

$ nano content/test.rst

Article title
#############

:date: AAAA-MM-JJ HH:MM
:modified: AAAA-MM-JJ HH:MM
:tags: word1, word2
:category: MyCategory
:slug: Article-title
:authors: MyName

Introduction ...

.. PELICAN_END_SUMMARY

Content ...

Generate the web site

$ make html

Preview the web site

$ make serve
$ firefox localhost:8000

Publish the web site on GitHub

$ make github

Username for 'https://github.com': didier-bernaudeau
Password for 'https://didier-bernaudeau@github.com': *************************

How to create bootable USB flash drives with Rufus

Date dim. 12 juillet 2015 Modified mer. 15 juillet 2015 Tags Windows / Linux

This article describes how to create bootable USB flash drives from ISO image with Rufus. The advantages of this software is that it supports UEFI & GPT and can create installation media for Microsoft Windows 7/8. The disadvantage is that it runs only on Microsoft Windows.

Install Rufus

Download the software on rufus.akeo.ie then execute the application.

  • Unpack the package
  • Copy the file "*.ucode" on another USB flash drive

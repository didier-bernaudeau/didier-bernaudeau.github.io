Laptop - Sony VAIO VGN SR11M

Date dim. 28 juin 2015 Modified mer. 15 juillet 2015 Tags Hardware

This article describes the hardware specification and configuration of my Sony VAIO VGN SR11M based on Intel Core 2 Duo processor. Future articles will refer to this one.

Hardware specification

Processor

Processor family Intel Core 2 Duo
Processor model P8400
Processor architecture Penryn 3M
Processor frequency 2.26 GHz
Processor Cache L2 3 MB
FSB (Front Side Bus) 1066 MHz

Chipset

Northbridge model Intel GM45 Express
Northbridge code name Cantiga
Southbridge model ICH9-M

Memory

Internal memory 3 GB (1 GB + 2GB)
Internal memory type DDR2-SDRAM
Memory clock speed 800 MHz
Maximum internal memory 4 GB

Graphics

Graphics Processor Unit Intel GMA X4500MHD
Maximum graphics memory 1300 MB

Audio

Audio controller Intel HD Audio
Audio codec Realtek ALC262
Number of speakers 2

Display

Display diagonal 33,78 cm (13.3")
Display resolution 1280 x 800 pixels

Hard Disk Drive

Hard drive model Fujitsu MHY2250BH
Total storage capacity 250 GB
Hard drive interface Serial ATA
Hard drive speed 5400 RPM

Optical Disc Drive

CD-R Read: 24 x / Write: 24 x
CD-RW Read: 24 x / Write: 16 x
DVD-R Read: 8 x / Write: 8 x
DVD-RW Read: 8 x / Write: 6 x
DVD-R Double Layer Read: 8 x / Write: 4 x
DVD+R Read: 8 x / Write: 8 x
DVD+R Double Layer Read: 8 x / Write: 4 x
DVD+RW Read: 8 x / Write: 8 x
DVD-RAM Read: 5 x / Write: 5 x

Ethernet

Model Marvell Yukon 88E8040
Standard Fast Ethernet (100 Mbps)
Interface PCI Express 1x

Wifi

Model Intel WiFi Link 5100
Standard 802.11 a/b/g/draft n
Interface PCI Express 1x

Bluetooth

Controller name Alps Electric BCM2046
Controller interface USB
Supported version v2.0 + EDR

Modem

Modem speed 56 Kbit/s
Modem type V.90/V.92

FireWire Controller

Model Ricoh R5C832
Interface PCI Express 1x

Card reader (Memory Stick)

Model Ricoh R5C592
Interface PCI Express 1x

Card reader (Secure Digital)

Model Ricoh R5C822
Interface PCI Express 1x

Camera

Camera model Ricoh VGP-VCC9
Camera resolution 1,3 MP
Video capturing speed 30 fps
Camera interface USB

Ports & interfaces

USB 2.0 x 2
VGA (D-Sub) x 1
Headphone outputs x 1
Microphone in x 1
Ethernet LAN (RJ-45) x 1
Modem (RJ-11) x 1
Firewire (IEEE 1394) x 1
ExpressCard slot x 1
Docking connector Yes

Hardware architecture

BIOS

Version

The last version is "R0310Y1"

Configuration

Press F2 to enter in BIOS menu.

Menu "Advanced"

Intel(R) Virtualization Technology [Enabled]

Menu "Boot"

Select 1st Boot Priority [External Device]
Select 2nd Boot Priority [Internal Optical Disc Drive]
Select 3rd Boot Priority [Internal Hard Disk Drive]

Jenkins

Date sam. 21 janvier 2017 Modified sam. 21 janvier 2017 Tags Development tool / Debian

Jenkins is the main tool for continuous integration and continuous delivery. Indeed, this tool is in the center of a development pipeline because it is able to interact with a variety of tools (Git repository, source code analysis, ...). Jenkins, a fork of Hudson, is a open source software written in Java.

Debian Stretch

  • Install Debian
Hostname: jenkins
Domain name: intra.net
  • Install sudo
# apt-get install -y sudo
# adduser UserName sudo
  • Update system
# sudo apt-get update && sudo apt-get upgrade

Jenkins

  • Add the key of jenkins' repository
$ wget -q -O - https://pkg.jenkins.io/debian/jenkins.io.key | sudo apt-key add -
  • Add the repository
$ sudo nano /etc/apt/sources.list

deb https://pkg.jenkins.io/debian binary/
  • Install apt module
sudo apt-get install apt-transport-https
  • Update the local package index
$ sudo apt-get update
  • Install Jenkins
$ sudo apt-get install jenkins
  • Create startup script for Jenkins
$ sudo nano /etc/systemd/system/jenkins.service

[Unit]
Description=Jenkins

[Service]
User=jenkins
Group=jenkins
WorkingDirectory=/var/lib/jenkins
ExecStart=/usr/bin/java -jar /usr/share/jenkins/jenkins.war
Restart=always

[Install]
WantedBy=multi-user.target
  • Start the service
$ sudo systemctl enable jenkins
$ sudo systemctl start jenkins

  • Open http://jenkins.intra.net:8080 in your browser to configure Jenkins

  • Get password

$ sudo cat /var/lib/jenkins/secrets/initialAdminPassword

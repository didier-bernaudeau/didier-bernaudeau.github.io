How to install Chromium with Flash support

Date ven. 23 octobre 2015 Modified ven. 23 octobre 2015 Tags Gentoo

In 2012, Adobe has stopped to upgrade Flash Player for Linux. So, the last version of Flash Player plugin is 11.2 that will be maintained until 2017. To continue to take the advantage of Flash content, we must find an alternative solution.

This tutorial describes how to install Chromium and the Flash Pepper Plugin (PPAPI) on Gentoo Linux and how to configure Chromium for graphic hardware acceleration.

Chromium

Define local USE flags

# nano /etc/portage/package.use
>=media-libs/libvpx-1.4.0       postproc

Install Chromium

# emerge -pv chromium

To force Chromium to use hardware acceleration, open chrome://flags and enable "Override software rendering list". To check if Chromium uses hardware acceleration, open chrome://gpu and look under "Graphics Feature status". All (or at least most of) the features should say "hardware accelerated".

Flash Player

To get Chrome and its plugins, download the RPM package for Linux 64bits on Google

Extract files from RPM package

$ rpmunpack google-chrome-stable_current_x86_64.rpm

Print the Flash Player version

# cat Chrome/google-chrome-stable_current_x86_64/opt/google/chrome/PepperFlash/manifest.json

Copy the Flash Player file

# cp Chrome/google-chrome-stable_current_x86_64/opt/google/chrome/PepperFlash/libpepflashplayer.so /usr/lib/chromium-browser/

Add Flash Player to Chromium

# nano /etc/chromium/default
CHROMIUM_FLAGS="--ppapi-flash-path=/usr/lib/chromium-browser/libpepflashplayer.so --ppapi-flash-version=19.0.0.207"

Malicious software

Date mer. 30 décembre 2015 Modified mer. 30 décembre 2015 Tags Software

This article indexes some malicious softwares (Mal-ware) : exploit kit, trojan, ...

Exploit Kit

An exploit kit is software system designed to run on web servers, with the purpose of identifying software vulnerabilities in client machines and exploiting discovered vulnerabilities to upload and execute malicious code on the client.

  • Blackhole
  • Rig
  • Angler

Ransomware

Ransomware is a type of malware that restricts access to a computer system and demands that the user pay a ransom to the operators of the malware to remove the restriction. Some forms of ransomware systematically encrypt files on the system's hard drive.

  • CryptoWall
  • kovter

Trojan

  • Bedep

Banker Trojan

Banker trojan is often used to steal banking information by man-in-the-browser (keystroke logging and form grabbing).

  • Zeus
  • Dridex
  • Cidrex
  • Gootkit

Botnet

  • Qakbot (aka Qbot)

TLS: Understanding the cipher suites

Date mer. 22 juillet 2015 Modified mar. 04 ao√ªt 2015 Tags Cryptography / Network

To protect your communications, TLS protocol uses 4 cryptographic functions: key exchange, authentication, encryption and message authentication code. Cipher suite allows to define the algorithm used for each of these 4 functions.

Cryptographic function

Key exchange

This cryptographic function is used to define a common secret (the master secret key) between two parties. Available algorithms are RSA, DH, DHE and ECDHE.

Authentication

This cryptographic function is used to authenticate the server by client and optionally the client by server. Available algorithms are RSA or DSS.

Encryption

This cryptographic function is used to encrypt data. Available algorithms are DES, 3DES, RC4, AES, ...

Message Authentication Code

This cryptographic function is used to sign data. Available algorithms are MD5, SHA, SHA256, ...

Cipher suites

Cipher suite is a list of one or more cipher strings separated by special char (colon, comma, space, ...). The names for cipher suites are defined by IANA (Internet Assigned Numbers Authority).

Cipher suite defines algorithms used for each cryptographic function: key exchange, authentication, encryption and message authentication code. If algorithm for key exchange is not mentioned, default algorithm is RSA.

The table below details some example of cipher suites:

Value Cipher Suite Key Exchange Authentication Encryption MAC
0x00,0x04 TLS_RSA_WITH_RC4_128_MD5 RSA RSA RC4 128 MD5
0x00,0x05 TLS_RSA_WITH_RC4_128_SHA RSA RSA RC4 128 SHA
0x00,0x0A TLS_RSA_WITH_3DES_EDE_CBC_SHA RSA RSA 3DES EDE CBC SHA
0x00,0x2F TLS_RSA_WITH_AES_128_CBC_SHA RSA RSA AES 128 CBC SHA
0x00,0x35 TLS_RSA_WITH_AES_256_CBC_SHA RSA RSA AES 256 CBC SHA
0x00,0x3C TLS_RSA_WITH_AES_128_CBC_SHA256 RSA RSA AES 128 CBC SHA 256
0x00,0x3D TLS_RSA_WITH_AES_256_CBC_SHA256 RSA RSA AES 256 CBC SHA 256
0x00,0x9C TLS_RSA_WITH_AES_128_GCM_SHA256 RSA RSA AES 128 GCM SHA 256
0x00,0x9D TLS_RSA_WITH_AES_256_GCM_SHA384 RSA RSA AES 256 GCM SHA 384

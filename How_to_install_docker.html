How to install Docker

Date jeu. 10 novembre 2016 Modified jeu. 10 novembre 2016 Tags Container / Debian

Docker is the most famous application container engine. This post describes the best way to install docker on Debian Linux.

Add Docker's repository

  • Install dependencies
# apt-get install apt-transport-https ca-certificates
  • Install public key of Docker's repository
# apt-key adv --keyserver hkp://p80.pool.sks-keyservers.net:80 --recv-keys 58118E89F3A912897C070ADBF76221572C52609D
  • Add Docker repository
# nano /etc/apt/sources.list.d/docker.list
deb https://apt.dockerproject.org/repo debian-jessie main
  • Update repository
# apt-get update
# apt-cache policy docker-engine

Install Docker

  • Install
# apt-get install docker-engine
  • Start Docker
# service docker start

Test Docker with OWASP ZAP

  • Pull the latest zap docker image from owasp's docker image repository (hosted by docker hub).
# docker pull owasp/zap2docker-stable
  • List installed images
# docker images
REPOSITORY                TAG                 IMAGE ID            CREATED             SIZE
owasp/zap2docker-stable   latest              a774bdc65502        5 months ago        1.557 GB
  • Run docker image
# docker run -p 8090:8090 -i owasp/zap2docker-stable zap.sh -daemon -port 8090 -host 0.0.0.0
  • Get Container ID
# docker ps

CONTAINER ID        IMAGE                     COMMAND                  CREATED             STATUS              PORTS                    NAMES
5ff0ca31a9a1        owasp/zap2docker-stable   "zap.sh -daemon -port"   54 seconds ago      Up 53 seconds       0.0.0.0:8090->8090/tcp   zen_knuth
  • Get information
# docker inspect <CONTAINER ID>

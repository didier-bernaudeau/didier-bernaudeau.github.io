MongoDB

Date sam. 09 juin 2018 Modified mar. 07 ao√ªt 2018 Tags MongoDB / Database

MongoDB is a document-oriented database very popular especially with MEAN technical stack (MongoDB, ExpressJS, AngularJS and NodeJS). This post will walk-through installing MongoDB on Gentoo Linux then it will list the most usefull command line with Mongo Shell tool (This second part could be a Cheat Sheet).

Server

# emerge mongodb 

# systemctl start mongodb

Now, MongoDB is listening on localhost interface on port 27017.

Create admin user

  • Connect to MongoDB server
$ mongo mongodb://localhost:27017
  • Create admin account
> use admin
> db.createUser(
  {
    user: "admin",
    pwd: "************",
    roles: [ { role: "userAdminAnyDatabase", db: "admin" } ]
  }
)
> exit
  • Enable authentication
# nano /etc/mongodb.conf

security:
    authorization: "enabled"
  • Restart MongoDB
# systemctl restart mongodb
  • Connect to MongoDB server with admin account
$ mongo mongodb://localhost:27017

> use admin
> db.auth("admin", "************")
  • Create simple user account on MyDB
> use MyDB
> db.createUser(
  {
    user: "MyDbUserAccount",
    pwd: "************",
    roles: [ { role: "readWrite", db: "MyDB" } ]
  }
)

Other useful command lines for Mongo Shell

  • Connection to MongoDB server
$ mongo
  • List of existing databases
> show databases
  • Create a new database and a collection
> use mydb
> db.MyCollection.insert({"name":"didier"})
  • List of existing collections
> show collections
  • Create an index
> db.MyOtherCollection.createIndex( { id: 1 } )
  • Print all documents
> db.MyCollection.find( {} )
  • Remove a document
> db.MyCollection.remove( {"_id": ObjectId("4d512b45cd9374271b02ec4f")})

HowTo configure the Linux kernel for Lenovo ThinkPad X240

Date mar. 16 juin 2015 Modified dim. 21 juin 2015 Tags HowTo / Gentoo / ThinkPad X24O

This article describes step by step how to configure the Linux Kernel for my Lenovo ThinkPad X240. This kernel must support Intel Haswell processor, all devices and UEFI bootloader without Grub.

To get more information about the configuration of my Laptop, you can read this article : Lenovo ThinkPad X240

Kernel installation

Download kernel Gentoo sources (3.18.12) :

# emerge sys-kernel/gentoo-sources sys-firmware/iwl7260-ucode

Select kernel sources

# eselect kernel list
# eselect kernel set X
# cd /usr/src/linux

Print configuration menu

# make menuconfig

Kernel compilation

# make && make modules_install
# cp arch/x86_64/boot/bzImage /boot/kernel-3.18.12

Kernel installation

Mount ESP (EFI System Partition)

# mkdir /mnt/ESP
# mount /dev/sda2 /mnt/ESP

Create directory

# mkdir /mnt/ESP/EFI/GentooLinux

Copy kernel

# cp /usr/src/linux/arch/x86_64/boot/bzImage /mnt/ESP/EFI/GentooLinux/3.18.12.efi

Unmount ESP (EFI System Partition)

# mount /mnt/ESP

Update UEFI bootloader

Install EFI boot manager

# emerge efibootmgr

Add the new kernel

# efibootmgr -c -d /dev/sda -p 2 -L "Gentoo Linux 3.18.12" -l "\EFI\GentooLinux\3.18.12.efi"

Print the menu

# efibootmgr -v

Remove the previous kernel (after testing the new kernel)

# efibootmgr -b 15 -B

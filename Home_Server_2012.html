RAID

There is not a real hardware RAID controller on Intel Z68 chipset. Indeed, this is a "Fake RAID" which is a combination of hardware (Chipset) and software (driver). So, this is not possible to store ESP (EFI System Partition) on a RAID drive.

Operating System

  • Create USB boot media with Debian Stretch (This USB key will be a read only filesystem)
  • Copy firmware-realtek_20161130-3_all.deb on an another USB key (from https://packages.debian.org/stretch/all/firmware-realtek/download)
  • Start installation

Note: In UEFI-installer you must press e key instead Tab and use arrow keys for moving

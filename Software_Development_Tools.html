Software Development Tools

Date lun. 23 janvier 2017 Modified lun. 23 janvier 2017 Tags Development tool / Software

This article indexes some tools for software development (IDE, SCM, CI, repository manager, ...).

A periodic table of DevOps tools from https://xebialabs.com :

Integrated Development Environment

  • Eclipse
  • Orion (web-based IDE)
  • STS (Spring Tool Suite) based on Eclipse

Source code repository (SCM or CVS)

  • SVN (Subversion)
  • Git
  • Mercurial

Source code repository manager

  • GitHub
  • GitLab
  • Gogs

Continuous Integration

  • Jenkins
  • Travis CI
  • Team City

Build

  • Ant
  • Gradle
  • Maven

Artifact Repository

  • Bower
  • Docker
  • Maven
  • npm
  • NuGet
  • PyPI
  • Ruby Gems

Artifact Repository manager

  • Archiva by Apache
  • Artyfactory by JFrog
  • Nexus Repository Manager OSS by Sonatype

Source code analysis

  • Sonarqube (for Code Quality)
  • Checkmarx
  • Fortify by HP

Testing

  • Cactus by Apache (Deprecated)
  • FitNess
  • JUnit
  • Selenium HQ

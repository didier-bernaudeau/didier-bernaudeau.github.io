Create your first presentation with RevealJS

Date lun. 22 mai 2017 Modified lun. 05 juin 2017 Tags Development / JavaScript

RevealJS is a powerful framework for creating a modern and beautiful presentation in HTML5/JS/CSS. This post descirbes how to install RevealJS then how to make your first presentation.

Install needed software

  • Install nodeJS
# emerge nodejs
  • Install RevealJS
# npm i node-reveal -g

Create a presentation

  • Init a new presentation
$ reveal -i <project_name>
$ cd <project_name>

  • Edit index.html file to add content

  • Start server in watch mode

$ reveal -w
  • Then, open your browser :
    • Multiplex presentation: http://localhost:3000/?role=master#/
    • Export presentation: http://localhost:3000/?print-pdf#/

Customize your presentation

  • First slide
    <style>
      .FirstSlides{
          width: 1280px;
          height: 720px;
      }
      .Title{
        position:absolute;
        right:2%;
        width:70%;
        top:20%;
        background-color:rgba(46,89,82,0.5);
        text-align: right;
      }
    </style>
...
    <section data-state="header1" data-background="img/background.jpg">
          <div class="FirstSlides">
            <div class="Title">
              <h1>Title</h1>
              <h4>Month Day<span class="ordinal">th</span>, 2017</h4>
              <p><small>By NAME</small></p>
            </div>
          </div>
    </section>

Docker Cheat Sheet

Date mer. 12 avril 2017 Modified mer. 12 avril 2017 Tags Container / Docker

This post is a quick reference sheet in one page for Docker

Container

  • Creates and start a container in one operation
# docker run -td -p 127.0.0.1:$HOSTPORT:$CONTAINERPORT -v $HOSTDIR:$DOCKERDIR

Options: -t allocates a pseudo-TTY session -d runs container in background -p maps the container port to the host port -v maps a directory on the host to a docker container

  • Show running containers
# docker ps

CONTAINER ID        IMAGE               COMMAND                  CREATED             STATUS              PORTS                    NAMES                     
f91ce070d170        postgres:9.6        "docker-entrypoint.sh"   2 minutes ago       Up 2 minutes        0.0.0.0:5432->5432/tcp   stupefied_easley
  • Stop a running container
# docker stop

Image

  • Show all images
# docker images

Registry & Repository

  • Pull an image from registry to local machine.
# docker pull ImageName
  • Push an image to the registry from local machine.
# docker push

Network

# docker network ls

Debian Stretch

Date jeu. 02 février 2017 Modified ven. 03 février 2017 Tags Debian

Stretch is the 9th version of Debian Linux and succeed to Debian Jessie. This version is still in testing, but the transition freeze stage will begin on February 5th 2017 and should become stable in September 2017. This tutorials is a guideline to install and secure Debian Stretch (RC2).

Installation

  • Download Debian Stretch ISO
cd /var/lib/libvirt/images/iso/
$ wget http://cdimage.debian.org/cdimage/stretch_di_rc2/amd64/iso-cd/debian-stretch-DI-rc2-amd64-netinst.iso
  • Create a virtual machine with Libvirt, QEMU and KVM
$ virt-install \
--connect qemu:///system \
--name=debian.intra.net \
--os-type=linux \
--os-variant=debiantesting \
--vcpus=1 \
--ram=1024 \
--graphics spice \
--network network=VirtualSwitch,model=virtio \
--boot uefi \
--disk path=/var/lib/libvirt/images/disk/debian.qcow2,format=qcow2,size=10,bus=virtio \
--cdrom=/var/lib/libvirt/images/iso/debian-stretch-DI-rc2-amd64-netinst.iso

  • Follow the instruction on the screen

  • Install sudo

# apt-get install -y sudo
# adduser UserName sudo
  • Update system
# sudo apt-get update && sudo apt-get upgrade

What still needs to be done

  • Install a Firewall IPv4/v6
  • Install SeLinux
  • Secure Boot

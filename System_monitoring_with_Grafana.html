System monitoring with Grafana

Date lun. 16 octobre 2017 Modified lun. 19 f√©vrier 2018 Tags APU / Debian Stretch

Grafana is an alternative to Chronograf in a TICK stack (Telegraf, InfluxDB, Chronograf, Kapacitor) and allows to create more beautiful dashboard than Chronograf. So, we could call that "TIGK" (Telegraf, InfluxDB, Grafana, Kapacitor). This post walks you through installing Grafana on Debian Stretch on which Telegraf and InfluxDB are already installed.

Add additional repository

  • add Grafana secret key (from Package Cloud)
$ curl -sL https://packagecloud.io/gpg.key | sudo apt-key add -
  • Add Grafana repository
$ echo "deb https://packagecloud.io/grafana/stable/debian/ stretch main" | sudo tee /etc/apt/sources.list.d/grafana.list

Grafana

  • Install Grafana
$ sudo apt-get install grafana
  • Configure firewall
$ sudo iptables -A INPUT -i $LanInterface -d $LanAddress -p tcp --dport 3000 -m conntrack --ctstate NEW,RELATED,ESTABLISHED -j ACCEPT
  • Start Grafana
$ sudo systemctl start grafana-server

  • Open Grafana in your browser http://g8way.domain.net:3000

  • Default user: admin

  • Default password: admin

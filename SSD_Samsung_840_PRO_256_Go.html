SSD - Samsung 840 PRO (256 Go)

Date sam. 18 juillet 2015 Modified sam. 18 juillet 2015 Tags Hardware

This article describes the new SSD Samsung 840 PRO 256 Go. I installed this SSD in my laptop Lenovo ThinkPad X240 to replace the original HDD.

Hardware specification

Model code MZ-7PD256BW
Capacity 256.00 GB (1 GB = 1 Billionbyte)
Capacity (Real) 238.47 GB
NAND flash memory Samsung Toggle DDR 2.0 NAND Flash Memory (MLC)
Controller Samsung 3-core MDX controller
Cache memory Samsung 512 MB Low Power DDR2 SDRAM
Interface SATA 6 Gb/s
TRIM Supported
S.M.A.R.T. Supported
Garbage Collection Supported
Encryption Support AES 256-bit FDE (Full Disk Encryption)
Firmware DXM06B0Q
Form factor 2.5 inch
Dimension (LxWxH) 100 x 69.85 x 6.8 (mm)
Weight 54 g
warranty 5 years

Performance

Sequential read Max. 540 MB/s
Sequential write Max. 520 MB/s
4KB Random Read (QD1) Max. 9,900 IOPS
4KB Random Write(QD1) Max. 31,000 IOPS
4KB Random Read(QD32) Max. 100,000 IOPS
4KB Random Write(QD32) Max. 90,000 IOPS

Partition Table

This partition table is created in GPT (GUID Partition Table) during Microsoft Windows 8.1 installation process

# Size Type File System Description
1 300 MB Basic Data Partition NTFS Windows 8.1 - Recovery
2 100 MB EFI System Partition FAT32 Windows 8.1 - EFI
3 128 MB Microsoft Reserved Partition NTFS Windows 8.1 - Reserved
4 49,5 GB Basic Data Partition NTFS Windows 8.1
5 8 GB Linux Swap SWAP Linux Swap
6 50 GB Linux filesystem EXT4 Linux Gentoo
7 130.5 GB Microsoft basic data FAT64 Data

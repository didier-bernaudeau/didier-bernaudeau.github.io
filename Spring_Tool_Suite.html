Spring Tool Suite

Date dim. 26 mars 2017 Modified dim. 26 mars 2017 Tags Java / Spring

STS (Spring Tool Suite) is an IDE (Integrated Development Environment) provided by Pivotal. STS is an Eclipse-based that is customized to develop application based on Spring framework. This tutorial describes how to install Spring Tool Suite and additional framework provided by Pivotal.

Spring Tool Suite

  • Install maven
# emerge dev-java/maven-bin
  • Download Spring Tool Suite
$ wget http://download.springsource.com/release/STS/3.8.3.RELEASE/dist/e4.6/spring-tool-suite-3.8.3.RELEASE-e4.6.2-linux-gtk-x86_64.tar.gz
  • Install Spring Tool Suite
$ tar xvzf spring-tool-suite-3.8.3.RELEASE-e4.6.2-linux-gtk-x86_64.tar.gz
$ mv sts-bundle spring-tool-suite

warning: the user must have write access in this directory. So, you can't install this software in "opt" directory for all users.

Spring ROO

  • Download Spring ROO
$ wget http://spring-roo-repository.springsource.org.s3.amazonaws.com/release/ROO/spring-roo-1.3.2.RELEASE.zip
$ unzip spring-roo-1.3.2.RELEASE.zip -d /mnt/data/spring-tool-suite/

  • Open STS and click on "Extensions" bottom tab and search "Spring Roo".

  • Select "Spring Roo (current production release)" and "Spring IDE - (Roo extension)"and follow the instructions

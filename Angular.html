Angular

Date ven. 11 mai 2018 Modified ven. 11 mai 2018 Tags Angular / JavaScript / Front-End

Angular is an open source framework from Google which allows developers to create single page applications. Angular has become a common framework used by many front-end components, event though this framework is very difficult to learn with a "steep learning curve". So, in this post, I summarize what I learnt on this framework. Angular should not be confused with AngularJS. This post concerns only Angular (version 2 to 6).

Architecture

  • Module: an angular module is set of angular basic building blocks like components, services etc. An application can have more than one module. A module can be created using @NgModule decorator.

  • Component: a component is one of the basic building blocks of an Angular application. An application can have more than one component. In a normal app, a component contains an HTML view page class file, a class file that controls the behaviour of the HTML page and the CSS/scss file to style your HTML view.

Routing

Angular Router allows to map a URL path to a component.

  • Import Angular Router (app.module.ts)
import { RouterModule, Routes } from '@angular/router';
  • Define route (app.module.ts)
const appRoutes: Routes = [
  { path: 'things',     component: ThingsListComponent },
  { path: 'thing/:id',  component: ThingDetailComponent }
];
  • Apply route to the module (app.module.ts)
@NgModule({
  imports: [
    RouterModule.forRoot(
      appRoutes,
      { enableTracing: true } // <-- debugging purposes only
    )
    // other imports here
  ],
  ...
})
  • Add a navigation bar in your root component template (app.component.html)
<nav>
    <a routerLink="/articles" routerLinkActive="active">Articles List</a>
    <a routerLink="/article/4" routerLinkActive="active">Article n4</a>
</nav>
<router-outlet></router-outlet>

Data binding

Data-binding in Angular apps is the synchronization of data between the model and view. There are various ways in which you can make use of data binding in the Angular framework.

Interpolation

You bind a property name in your view template by making use of double curly braces {{ value }}.

Ng bind

The ng-bind-html directive is a secure way of binding content to an HTML element. => Output encoding

Hello, <span ng-bind="value"></span>

It seems to be twice as fast to use ng-bind than interpolation.

Create a service (with Angular 5.2)

  • Create the service my.service.ts
import { Injectable } from '@angular/core';

@Injectable()

export class MyService {

}
  • Import service in app.module.ts
...
import { MyService } from './my.service';
...

@NgModule({
  ...
  providers: [
   MyService
  ],
  ...
})
  • Use this service in app.component.ts
...
import { MyService } from './my.service';
...

export class AppComponent {
  constructor(private _MyService:MyService) { }
}

  providedIn: 'root',
